Tag => tags are come in between < AngularBrackets>

We have many tags in html for eg. h1, p, header, div, span etc.

90% tags have start tag and end tag. but some tags are self closing eg. br, hr, img, input etc. They are also called as Void Tag or Empty Tags

Elements => Start Tag - Some Content - End Tag

Attributes => Elements in HTML have attributes; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want.
